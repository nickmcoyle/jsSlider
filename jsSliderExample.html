Example jsSlider

While a lot of people are making sliders, they tend to use a lot of frameworks, and I just wanted to make my own to see how hard it would be to make one from scratch using only plain javscript.

Goals for this project

  • dont use any frameworks, only vanilla javascript
  • try to make it work down to at least ie8 and try to make it very fast so the user doesn't notice any lag or jerkiness in the slider
  • make it responsive so it works on mobile too
  • use vector graphics for buttons and dots so they will scale well
  • make it a plugin so anyone can use it on their webpage by including my js and css files and calling a constructor

 

What you will need

  • download the css and js files in this repository and include them in your webpage
  • insert a div into your HTML with id="sliderContainer"
  • write 2 lines of javascript on your page, one line creates a new instance of the slider, the 2nd line instantiates it
